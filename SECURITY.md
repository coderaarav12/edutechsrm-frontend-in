# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in edutechsrm, please report it privately by emailing **admin@edutechsrm.in**.

Do not open a public GitHub issue — I'll investigate and address it as soon as possible.

## What to include

- A clear description of the issue
- Steps to reproduce (if applicable)
- Any relevant logs, screenshots, or proof-of-concept

## Scope

This policy covers the edutechsrm frontend application and its deployed Cloudflare Worker. For issues in the backend API, report them to the same email.

## Response timeline

I aim to acknowledge reports within 48 hours and release a fix as soon as the issue is understood and resolved.
